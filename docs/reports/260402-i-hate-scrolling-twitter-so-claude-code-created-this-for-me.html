I Hate Scrolling Twitter, So Claude Code Created This For Me

Video · AI & Technology · 2 Apr 2026 · 1m · source

⚡ BOTTOM LINE

A custom-built Twitter bot created with Claude Code automates the discovery of trending AI content using a five-signal ranking system and a self-correcting feedback loop, but the complete solution is sold commercially as part of Chase AI Plus.

📝 THESIS

The speaker describes an automated system that scrapes, filters, and ranks viral AI tweets every 30–45 minutes, delivering the top pick to Telegram via a pipeline built with Claude Code and integrated with AI response grading to continuously improve relevance.

💡 KEY INSIGHTS

  1. Five‑Signal Ranking — Tweets are scored on velocity (likes/retweets growth), authority (poster credibility), timing, opportunity, and replyability; the top candidate is selected via a softmax probability distribution1. [✓]

  2. Self‑Correcting Feedback — All user responses are stored in Supabase and graded by AI combined with the user’s own rating, creating a loop that refines both tweet selection and response style over time1. [✓]

  3. Commercial Stack — The system uses Claude Code (referred to as “Cloud Code” in the transcript), Apify (likely the “Ampify” scraper), Supabase (called “Superbase”), Grok (mis‑transcribed as “Grock”), Railway, and Vercel, and is offered via the Chase AI Plus subscription1. [✓]
