5. **Claude Code Artifacts turn sessions into live, shareable pages** – Artifacts capture code, test results and reasoning in a persistent URL, auto‑updating for teammates in real time<sup>[2]</sup>.

7. **Codex Record & Replay automates Mac workflows** – Recording a UI sequence creates a reusable AI skill, rivaling RPA tools like UiPath with virtually no setup, currently for Mac users on paid Codex plans<sup>[6]</sup>.
